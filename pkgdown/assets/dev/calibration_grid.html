Headline reads

  1. Multinomial coverage is zero across every cell. pigauto's 3-class credible set collapses to one class with ≥95% mass; when wrong, truth is never in the set. Matches BACE's over-confident- classifier finding.
  2. Binary coverage ~0.25. Same over-confidence story.
  3. Gaussian coverage ~0.30. Consistent with v0.9.1's min_val_cells warning (conformal degrades when val_cells < ~20).
  4. Poisson ~0.53 / Ordinal ~0.67. Partially covered; integer support makes MI quantiles naturally wider.
  5. trait_MNAR binary ~0.03. Pathological regime as theory predicts.

Future work

  • Rerun at larger n (300, 500, 1000) to chart the small-n regime boundary.
  • Also compute conformal coverage (not in this grid); the theoretical guarantee should pull it to 0.95 at larger n.
  • Issue #40's median-pool fix is orthogonal (fixes point estimate, not coverage).
