Coverage calibration grid (v0.9.1.9000)

3,000 fits on BACE-generated simulated data: 4 signal scenarios x 3 missingness mechanisms x 5 trait types x 50 reps, n=150 per fit. Target coverage = 0.95 (MC-dropout credible set).

Full report. Headline: conformal intervals hit the 95% guarantee; MC-dropout is over-confident at n=150 (gaussian ~0.30, binary ~0.19, multinomial ~0.00, poisson ~0.52, ordinal ~0.67). Replicates BACE paper's small-n over-confidence finding.

How to read the Improvement column. For RMSE metrics, positive values ( ▲) mean pigauto achieved lower error than the baseline. For accuracy metrics, positive values mean pigauto achieved higher accuracy. A value near zero is expected when phylogenetic signal dominates and the calibrated gate closes to zero — in that case pigauto falls back to the baseline by design.
